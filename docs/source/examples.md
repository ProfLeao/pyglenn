# Examples

Collection of Jupyter *notebooks* demonstrating how to use **pyglenn**. The
notebook source files live in [`examples/`](https://github.com/ProfLeao/pyglenn/tree/main/examples)
at the repository root and can be run locally with `jupyter lab examples/`.
